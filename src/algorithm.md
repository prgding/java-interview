## 排序算法

### 冒泡 O(n²)

- 原理
    1. 遍历比较相邻，如果发现左 > 右，则交换位置
    2. 经过一次遍历后，最大的元素会“冒泡”到最右边
    3. 重复这个过程，但忽略右边排序好的

- 实现


```java
public class Sort{
    public static void bubbleSort(int[] arr){
        for (int i = 0; i < arr.length - 1; i++) {
            for (int j = 0; j < arr.length - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }
}
```
